Html Boilerplate

El Html Boilerplate es la estructura que nos sirve como plantilla para la creación de páginas web.

Del H1 al H6

Es una etiqueta que sirve para introducir títulos de diferente tamaño. Ejemplo:

-Escrito con h4

Eitqueta br

Es una etiqueta que sirve para realizar un salto de línea cuando uno quiera. Por ejemplo:
Escribimos una oración y queremos un salto aquí
para escribir en otra línea.

Escribir en cursiva

Para escribir en cursiva tienes que utilizar la etiqueta em. Un ejemplo:
Esto está escrito en cursiva

Escribir en negrita

Para escribir en negrita debes usar la etiqueta strong. Por ejemplo:
Esto está escrito en negrita

Listas

Desordenadas

Listas organizada por puntos. Se utiliza la etiqueta ul si se quieren añadir más objetos a la lista se utiliza la etiqueta li. Por ejemplo:

  • Leche
  • Huevos

Ordenadas

Listas organizadas en un nivel de importancia por números. Se utiliza la etiqueta ol y si se quieren añadir más objetos a la lista se utiliza la etiqueta li. Un ejemplo:

  1. Messi y Cristiano
  2. Lewandowski y Benzema
  3. Mbappé y Haaland

Insertar imágenes

Desde internet

Para insertar una imagen en la página web que esté almacenada en un sitio web de internet debemos utilizar la etiqueta img y el atributo src=url de la imagen que queramos utilizar. Si el sitio web en la que está alojada no funciona, la imagen no se verá. En cambio se mostrará un texto alternativo que podemos elegir nosotros con el atributo alt.

En almacenamiento local

Para insertar imágenes en la página web que estén almacenadas en nuestro ordenador de forma local debemos utilizar la etiqueta source y poner la ruta donde se encuentra esa imagen. También puedes utilizar la etiqueta alt para poner un texto alternativo.
