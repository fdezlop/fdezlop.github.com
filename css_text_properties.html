Propiedades de texto de CSS

Font Family genérica

Una familia de fuente genérica es una familia de fuente en general, por ejemplo sans-serif o serif.
Por ejemplo, este párrafo esta escrito con sans-serif, una familia de fuente genérica.

Familia de fuente específica

Una familia de fuente específica es un tipo de letra concreto que pertenece a una familia determinada como por ejemplo helvetica. Estos se ponen entre comillas dentro de la etiqueta style.
Este párrafo está escrtio con la letra courier, un tipo de letra de la familia de fuente serif

Tamaño de fuente (px)

Para cambiar el tamaño de fuente debemos utilizar el atributo font-size dentro de la etiqueta style. A este le daremos un valor en píxeles (px)
Por ejemplo, este párrafo esta escrito con una letra de 15 px.

Tamaño de fuente (em)

Tambień podemos cambiar el tamaño utilizando em en vez de px. Em sirve para poder modificar el tamaño con unas proporciones.
Por ejemplo, este texto está escrito con una fuente un 150% más grande que lo predeterminado por la página.

Estilo de fuente

El estilo de fuente se puede cambiar utilizando el atributo font-style dentro de una etiqueta style. En ella podemos poner la letra en cursiva, utilizando italic.
Como ejemplo, este párrafo está escrito en cursiva.

Altura entre líneas

La altura entre líneas se puede modificar utilizando el atributo line-height dentro de la etiqueta style.
Por ejemplo, la altura entre líneas de este párrafo es de 20px.

Alinear texto

Para alinear un texto a la izquierda, al centro o a la derecha tendrás que utilizar el atributo text-align dentro de la etiqueta style. Para alinearlo a la derecha, pondras text-align: right, y si lo quieres alinear al centro text-align: center.
Por ejemplo, este texto esta alineado al centro y el título está alineado a la derecha.

Decoración del texto

Para decorar el texto tienes que usar el atributo text-decoration dentro de la etiqueta style.
