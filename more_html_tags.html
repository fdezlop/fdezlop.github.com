Enlace desde texto

Para poner un enlace en un texto deberemos utilizar la etiqueta a, y dentro de esta deberemos poner el texto que queramos que se convierta en un enlace.
Por ejemplo: Este texto te lleva a la página del instituto.

Enlace desde una imagen

Para poner un enlace en una imagen debemos utilizar la etiqueta a y dentro de esta insertar la imagen en la que queramos poner el enlace con la etiqueta img

Enlace absoluto desde un texto o imagen

Un enlace absoluto es aquel que nos dirige a una página web ajena a nuestro sitio web. Se insertan igual que otro enlace.
Por ejemplo, los enlaces anteriores son absolutos, puesto que te dirigen a una página web diferente al sitio web.

Enlace relativo desde un texto o imagen

Un enlace relativo es aquel que te dirige a una página web alojada en el sitio web en el que se encuentra el enlace. Se insertan igual que otro enlance.
Por ejemplo, los enlaces ubicados en el index y al final de cada página web de mi sitio web son relativos.

Enlace que se abre en una ventana nueva

Para que un enlace se abra en una ventana nueva debes utilizar la etiqueta a con el atributo target="_blank".
Por ejemplo, si pinchas aqui se te abrirá una nueva ventana con el enlace.

Un enlace interno a un sitio dentro de la misma página

Para insertar un enlace que te direccione a un sitio dentro de la misma página web debes utilizar la etiqueta a con el atributo href="#id". El enlace te direccionará al sitio que hayas definido con la id.
Por ejemplo, si pinchas aquí, el enlace te direccionará al principio de la página.

Insertar una tabla

Para insertar una tabla en tu página web, debes utilizar la etiqueta table. Después, tenemos que utilizar la etiqueta thead (para la cabeza de la tabla) y tbody (para el cuerpo de la tabla). Para añadir lineas, tendremos que utilizar la etiqueta th (para añadir columnas en el encabezado de la tabla) y td (para añadir filas en el cuerpo).
Por ejemplo, la siguiente tabla:

Nombre Equipo Dorsal
Lionel Messi Paris Saint-Germain 30
Cristiano Ronaldo Manchester United 7
Erling Haaland Borussia Dortmund 9
