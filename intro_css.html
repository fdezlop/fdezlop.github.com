Seleccionando por etiqueta

Se realiza creando una etiqueta style y poniendo el nombre de la etiqueta que queramos modificar, por ejemplo, p, abrimos corchetes y dentro ponemos los atributos.

  • Si queremos cambiar el color, dentro de los corchetes deberemos poner color:rgb(x,y,z), o bien un color común así color: yellow;.
  • Si queremos modificar el color del fondo debermos realizar los pasos anteriores pero donde pone color lo cambiamos por background-color.
Por ejemplo, en esta página web todos los h2 están escritos en rojo con un fondo negro.

Seleccionando por id

Para selecionar por id debemos emplear la mima estructura, pero debemos poner # antes del id. El id se lo daremos a una etiqueta en concreto, y lo definiremos así: id="...".Por ejemplo:

-Esta frase esta definida por un id por Marcos Fernández.

Seleccionando por clase

Para seleccionar por clase debemos poner un punto (.) antes del nombre que le hayamos dado a la clase. La clase se la podemos dar a varias etiquetas y estas se verán afectadas. Para ello, dentro de una etiqueta pondremos class="...". Por ejemplo:

-Los parrafos de este apartado y el título principal tienen una misma clase.
